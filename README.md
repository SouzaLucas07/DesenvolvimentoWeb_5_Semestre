# DesenvolvimentoWeb_5_Semestre
Repositório com projetos e atividades da disciplina de Desenvolvimento Web do 5º semestre de Engenharia da Computação da FACENS. Contém práticas voltadas à arquitetura web, desenvolvimento front-end e back-end, versionamento com Git e aplicação de boas práticas de engenharia de software na construção de aplicações web modernas.
